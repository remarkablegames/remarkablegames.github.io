[Kaplay Template](https://github.com/remarkablegames/kaplay-template) allows you to make web games in [Kaplay.js](https://kaplayjs.com/).

## Motivation

This [template](https://github.com/remarkablegames/kaplay-template) allows you to bootstrap a [KAPLAY](https://kaplayjs.com/) game.

The [repository](https://github.com/remarkablegames/kaplay-template) contains:

- a simple scene with a player that you can control
- [GitHub Actions](https://github.com/features/actions) to:
  - deploy the web build to [GitHub Pages](https://pages.github.com/)
  - release the game to [itch.io](https://itch.io/)
  - build a desktop app with [tauri-action](https://github.com/remarkablemark/tauri-action)
