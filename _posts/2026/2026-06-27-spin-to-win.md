## Background

I made this game for [The Very Serious Juniper Dev Game Jam](https://itch.io/jam/theveryseriousjuniperdevgamejam), and the theme was **spin to win**.

I built the game using [kaplay-template](https://github.com/remarkablegames/kaplay-template) and implemented features such as:

- 10 levels, each with 3 rounds and a target score
- Spin the wheel to land on positive segments (score, money, and multipliers) while avoiding penalties
- Spend money in the shop to add and upgrade wheel segments
- Buy artifacts that influence future spins

Check out the source code on [GitHub](https://github.com/remarkablegames/spin-to-win) and give the game a [spin](https://remarkablegames.itch.io/spin-to-win)!
